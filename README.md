# Personal Portfolio Website

A Full Stack Personal Portfolio Website built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Features

- Responsive Design
- About Section
- Projects Section
- Contact Form
- MySQL Database Storage
- REST API

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/personal-portfolio-website.git
```

### Install Packages

```bash
npm install
```

### Create Database

Import:

```bash
portfolio.sql
```

### Start Server

```bash
npm start
```

Server runs at:

```bash
http://localhost:5000
```
